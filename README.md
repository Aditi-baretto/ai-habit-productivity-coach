# 🧠 AI Habit & Productivity Coach

An AI-powered application that analyzes daily habits and gives productivity advice.

Built using **LangChain + Ollama + Streamlit** with a local LLM.

---

## 🚀 Features

✔ Habit analysis using AI
✔ Productivity score
✔ Habit visualization chart
✔ Motivational suggestions
✔ Interactive Streamlit UI

---

## 🛠 Tech Stack

* Python
* Streamlit
* LangChain
* Ollama
* Matplotlib

---

## 📂 Project Structure

ai-habit-productivity-coach
│
├── app.py
├── habit_analysis.py
├── prompts.py
├── requirements.txt
└── README.md

---

## ▶️ Run the Project

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
